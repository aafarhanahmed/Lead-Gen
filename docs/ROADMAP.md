# Lead-Gen Roadmap

## Current Goal

Turn the Python lead research workflow into a repeatable service engine for fresh local business prospect lists.

## Immediate Priorities

1. Make pipeline runnable from a single command.
2. Generate sanitized sample lead packs.
3. Create repeatable output columns.
4. Add deduplication and source URL tracking.
5. Create agency-ready sample assets.
6. Use tool to support paid pilots and agency retainers.

## Not in Scope

- LinkedIn scraping
- Sales Navigator scraping
- cold calling
- appointment setting
- PPC campaign management
- restricted-platform scraping
- guaranteed verified personal emails
